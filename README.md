# 🎙 台湾中国語 文字起こし＆添削ツール

Whisper でローカル音声認識 → Qwen2.5 で AI 分析・添削を行うブラウザアプリです。  
**データは外部に送信されません。すべてローカルで完結します。**

---

## 機能

- 🎬 動画・音声ファイルの文字起こし（MP4 / MOV / M4A など）
- 🔗 Google Drive 共有リンク・直接 URL からの文字起こし
- 📝 文字起こし済みテキストの直接貼り付け
- 🤖 AI による修正・添削・日本語訳（Qwen2.5 / Ollama）
- ▶️ 動画プレーヤー内蔵（タイムスタンプクリックでシーク）
- 🌏 混合言語モード（日本語・中国語を自動判別して分類表示）
- 🔒 完全ローカル処理

---

## 必要なもの

| ツール | バージョン | 用途 |
|--------|-----------|------|
| Python | 3.8 以上 | サーバー実行 |
| [Ollama](https://ollama.com) | 最新版 | AI モデル実行 |
| Qwen2.5:7b | — | 分析・添削モデル |

> Whisper は起動時に自動インストールされます。

## セットアップ

### 1. リポジトリをクローン

```bash
git clone https://github.com/masaa0802/taiwan-chinese-transcriber.git
cd taiwan-app
```

### 2. Ollama をインストールして起動

#### 方法A：アプリでインストール（コマンドライン不要）

**① アプリをインストール**

1. [https://ollama.com](https://ollama.com) を開き、`Download` をクリック
2. **Mac** の場合：ダウンロードした `.zip` を解凍 → `Ollama.app` をアプリケーションフォルダにドラッグ → 起動  
   **Windows** の場合：ダウンロードした `OllamaSetup.exe` をダブルクリックしてインストール → 起動
3. メニューバー（Mac）またはタスクトレイ（Windows）にラマのアイコンが表示されれば OK  
   → `ollama serve` は**自動で起動**するため、コマンドを打つ必要はありません

**② Qwen2.5 モデルをダウンロード（初回のみ・約 4.7 GB）**

ターミナルを一度だけ開いて以下のコマンドを貼り付けて実行します。

- **Mac**：`Cmd + Space` を押して「ターミナル」と入力 → Enter で起動
- **Windows**：スタートメニューで「PowerShell」と検索 → 起動

```bash
ollama pull qwen2.5:7b
```

`success` と表示されたらターミナルは閉じて構いません。以降はコマンドを使う必要はありません。

---

#### 方法B：コマンドラインでインストール

```bash
# Ollama 公式サイト https://ollama.com からインストール後：
ollama serve
```

別ターミナルで Qwen2.5 モデルをダウンロード（初回のみ）：

```bash
ollama pull qwen2.5:7b
```

### 3. サーバーを起動

```bash
python3 server.py
```

初回起動時は Whisper モデル（約 150 MB）が自動でインストール・ロードされます。  
`✅ 準備完了！` と表示されると、ブラウザに `index.html` が自動で開きます。

## 使い方

### Step 1 — サーバー接続確認

起動時に自動で確認されます。「サーバー接続 OK」と表示されれば準備完了です。  
分析に使うモデルをドロップダウンで選択できます（デフォルト: `qwen2.5:7b`）。

### Step 2 — 動画を入力

**ファイル**タブ：MP4 / MOV / M4A などをドラッグ＆ドロップ、またはクリックして選択。

**URL**タブ：Google Drive の共有リンクまたは直接 MP4 URL を貼り付け。

**テキスト直接入力**タブ：他のツールで作成済みの文字起こし（SRT 形式など）を貼り付け。

言語を選択して「▶ 文字起こし」をクリック：

| 言語設定 | 説明 |
|----------|------|
| 中国語（自動） | 台湾の繁体字・普通話 |
| 広東語 | 香港・広東語 |
| 日本語 | 日本語のみ |
| 英語 | 英語のみ |
| **混合（日本語・中国語）** | 両言語が混在する動画（自動判別） |

### Step 3 — 結果を確認

**動画プレーヤー**：ファイルまたは URL から動画が自動で読み込まれます。  
**タイムスタンプをクリック**すると、その位置から動画が再生されます。

**混合言語モード**では文字起こし上部にフィルターボタンが表示されます：
- 「すべて」「中国語のみ」「日本語のみ」で絞り込み可能

### Step 4 — AI 添削（任意）

「🤖 AI 添削を実行」ボタンをクリックすると、Qwen2.5 が以下を分析します：

1. **修正・添削** — 誤認識の修正案と理由
2. **台湾中国語の特徴・解説** — 台湾特有の表現や大陸との違い
3. **日本語訳** — 全体の自然な日本語訳

## トラブルシューティング

**「サーバー未起動」と表示される**  
→ `python3 server.py` を実行しているか確認してください。

**文字起こしが遅い**  
→ Whisper `base` モデルは CPU でも動きますが数分かかる場合があります。

**Google Drive の動画が再生できない**  
→ サーバーが起動していることを確認してください。動画はサーバー経由でプロキシされます。

**AI 添削でエラーが出る**  
→ `ollama serve` が起動しているか、`qwen2.5:7b` がダウンロード済みか確認してください。
